# HOI4-Mods-modifiers_l_simp_chinese.yml-Fixer
用来批量补全钢丝mod中的buff汉化文件modifiers_l_simp_chinese.yml文本图标缺失的问题。
内含一个python文件'fix.py'和一组案例'modifiers_l_simp_chinese.yml'、'modifiers_l_simp_chinese_new.yml'和'modifiers_l_english.yml'。
其中'fix.py'是主程序，'modifiers_l_simp_chinese.yml'是未修改文件，'modifiers_l_simp_chinese_new.yml'是修改后文件，'modifiers_l_english.yml'是参考文件。
运行时文件夹内除主程序外必须包含'modifiers_l_simp_chinese.yml'和'modifiers_l_english.yml'，运行结束后自动生成'modifiers_l_simp_chinese_new.yml'，将其重命名为'modifiers_l_simp_chinese.yml'并覆盖原文件即可。
覆盖原文件前请做好备份。
